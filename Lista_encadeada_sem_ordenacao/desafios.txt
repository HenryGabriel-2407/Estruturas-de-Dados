1. Contador de Elementos (FINALIZADO)
Objetivo: Modificar a função mostrar para que, ao exibir a lista, também informe o total de elementos na lista.
Tarefa: Atualize a função mostrar para imprimir o número total de elementos ao final da exibição.

2. Inserir no Início
Objetivo: Implementar uma função para incluir um novo elemento no início da lista.
Tarefa: Crie uma função incluir_inicio que insere um novo nó no início da lista e atualize o início da lista.

3. Excluir Último Elemento
Objetivo: Implementar uma função para remover o último elemento da lista.
Tarefa: Crie uma função retirar_ultimo que remove o último elemento e atualiza o ponteiro fim da lista.

4. Remover Elementos Após um Valor
Objetivo: Modificar a função inclusao_apos para remover todos os elementos que seguem um determinado valor de referência.
Tarefa: Crie uma função remover_apos que remove todos os elementos que estão após um valor específico na lista.

5. Inverter Lista
Objetivo: Implementar uma função para inverter a ordem dos elementos na lista.
Tarefa: Crie uma função inverter_lista que altera a lista para que a ordem dos elementos fique invertida.

6. Buscar e Contar
Objetivo: Modificar a função de busca para contar quantas vezes um valor aparece na lista.
Tarefa: Atualize a função buscar para retornar a contagem de aparições do valor na lista.

7. Ordenar Lista
Objetivo: Implementar uma função para ordenar os elementos da lista em ordem alfabética.
Tarefa: Crie uma função ordenar_lista que organiza os elementos da lista em ordem crescente.

8. Adicionar Valor Antes de um Valor de Referência
Objetivo: Implementar uma função para inserir um novo elemento antes de um valor de referência.
Tarefa: Crie uma função incluir_antes que adiciona um novo elemento antes de um valor específico na lista.

9. Duplicar Lista
Objetivo: Criar uma função que duplica a lista ligada, ou seja, cria uma nova lista com os mesmos elementos.
Tarefa: Implemente uma função duplicar_lista que cria uma nova lista com todos os elementos da lista original.

10. Remover Todos os Elementos
Objetivo: Criar uma função para remover todos os elementos da lista, liberando a memória.
Tarefa: Crie uma função remover_todos que percorre a lista e exclui todos os nós, liberando a memória usada.
